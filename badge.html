DX 인증 배지 — 표준 snippet

아래 코드를 자신의 HTML 페이지에 박으면 우상단에 DX 인증 배지가 표시됩니다. 클릭하면 사내 허브로 이동해 진위 확인 가능.

중요: 이 배지는 DX팀이 인증한 도구에만 박을 수 있습니다. 인증 전 자기 도구에 임의로 박는 행위는 금지. 인증 신청은 DX 요청폼(P3 카테고리)으로.

1. CSS (head 안에 복사)

<style>
.dx-cert-badge {
  position: fixed; top: 12px; right: 12px; z-index: 9999;
  background: #1F4E79; color: white; padding: 6px 12px; border-radius: 20px;
  font-size: 12px; font-family: 'Malgun Gothic', sans-serif; font-weight: 600;
  text-decoration: none; box-shadow: 0 2px 6px rgba(0,0,0,0.15);
  display: inline-flex; align-items: center; gap: 6px; transition: transform 0.15s;
}
.dx-cert-badge:hover { transform: translateY(-1px); }
.dx-cert-badge .dot { width: 8px; height: 8px; background: #4CAF50; border-radius: 50%; display: inline-block; }
.dx-cert-badge .meta { opacity: 0.85; font-weight: 400; font-size: 11px; margin-left: 4px; }
</style>

2. HTML (body 안 어디든, 권장: 맨 위)

<a class="dx-cert-badge" href="https://tools.ogaren/?id=YOUR-TOOL-ID" target="_blank">
  <span class="dot"></span> DX 인증
  <span class="meta">v1.0 · 2026-05-26</span>
</a>

YOUR-TOOL-ID, 버전, 날짜는 DX팀이 인증 시 발급해서 알려드립니다. 직접 만들어 박으면 안 됩니다.

3. 미리보기

DX 인증 v1.0 · 2026-05-26

실제로는 페이지 우상단에 고정 표시됩니다. 우상단 한번 보세요 ↗

4. (선택) 자동 로더 버전 — 변조 방지 + 자동 업데이트

아래 한 줄만 박으면 허브에서 메타를 가져와 자동 렌더. 버전·날짜 직접 수정 불필요.

<script src="https://tools.ogaren/badge.js" data-tool-id="YOUR-TOOL-ID"></script>

⚠️ badge.js는 Phase 2에서 제공 예정 (Cloudflare Pages 배포 후). 현재 PoC에서는 위 정적 HTML로 사용.
