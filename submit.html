← 도구 허브

내 도구를 여기 등록하려면?

직원분들이 만들어 사용 중인 HTML 도구·대시보드를 도구 허브에 등록하고, 최신본임을 보장하는 DX 인증까지 받는 방법을 단계별로 안내합니다.

1이 허브가 뭔가요?

매일 우리 직원분들이 Claude·ChatGPT로 다양한 HTML 도구·대시보드를 만들어 사용하고 있습니다. 그런데 문제가 있죠.

  • v1, v2, v3가 매장마다 다 다름 — 누가 진짜 최신본인지 모름
  • 카톡으로 받은 파일이 진본인지 누가 임의로 수정한 건지 확인 불가
  • 좋은 도구가 다른 부서에 안 알려져 똑같은 걸 또 만듦

도구 허브는 이 문제를 해결합니다. 모든 사내 도구를 한 곳에 모으고, DX팀이 검토한 도구에는 우상단 "🟢 DX 인증" 배지를 자동으로 박아 "이게 진짜 최신·공식본"임을 누구나 확인할 수 있게 합니다.

23가지 등록 상태

도구는 허브에 다음 3가지 상태 중 하나로 표시됩니다.

DX 인증

정식 인증본

DX팀이 검토하고 정식 배포 경로를 거친 도구. 우상단에 인증 배지 자동 표시.

베타

검토 중 또는 부분 사용

DX팀이 만들었거나 검토 중이지만 아직 정식 인증 전. 담당 부서 책임.

개인본

등록만

직원분이 자체 제작하여 본인 책임으로 사용. 인증 아님 — 다른 사람에게는 권장 안 됨.

💡 인증 vs 등록만, 뭘 선택해야 할까요?

인증 신청은 다른 사람들도 안심하고 쓰게 하고 싶을 때 (5영업일 소요), 등록만은 일단 "이런 도구가 있어요" 알리고 싶을 때 (1~2일).

3어떤 도구가 등록 대상인가요?

✅ 등록 가능

  • Claude·ChatGPT로 만든 단일 HTML 파일 (대시보드, 조회 도구, 가이드 등)
  • Google Sheets + Apps Script로 만든 사내 시스템
  • Python·Excel 매크로로 만든 자동화 도구 (실행 결과가 HTML이면)
  • 여러 사람이 쓰거나, 본인만 쓰지만 좋은 노하우가 들어간 도구

❌ 등록 부적합

  • 일회성으로 한 번 쓰고 버리는 스크립트
  • 다른 도구와 너무 비슷해서 중복인 경우 (이 경우 DX팀이 통합 제안)
  • 회사 외부 사람을 위한 자료 (이건 다른 채널)

7자주 묻는 질문

Q. 인증받으면 코드가 GitHub에 공개되나요?

도구의 민감도에 따라 다릅니다. 일반 정보(가이드·조회 도구)는 GitHub Pages에 공개 배포되고, 부서 내부 정보·재무 데이터가 들어 있으면 Cloudflare Access로 사내 인증 도메인에 배포됩니다 — 직원 외에는 접근 불가. 폼/이메일의 "민감 정보 레벨"에 정확히 답해주시면 DX팀이 적절한 경로로 배포합니다.

Q. 인증 후 도구를 업데이트하면 어떻게 되나요?

본인이 새 버전을 만들면 DX팀에 알려주세요 (카톡 한 줄이면 됩니다). DX팀이 새 버전을 빌드 + 인증 배지에 새 버전 번호 표시 + 재배포합니다. 같은 URL이므로 사용자는 그대로 쓰면 자동으로 최신본을 보게 됩니다.

Q. 인증 거절될 수도 있나요?

네, 4가지 경우는 거절됩니다. ① 단발성이라 인증할 가치 낮음 ② 동일 기능 도구가 이미 인증되어 있어 중복 ③ 다른 사내 시스템과 충돌 ④ 보안·법적 이슈. 거절 시 사유를 명확히 알려드리고 보완 방향도 제안드립니다.

Q. 도구가 Excel·Sheets인데 인증 가능한가요?

Excel은 인증 모델에 안 맞아 등록만 가능합니다. Google Sheets + Apps Script는 인증 가능하지만 "배지 자동 박기"는 안 되고, 대신 시트 헤더에 인증 표식 + 허브 카드에 인증 상태 표시로 처리됩니다. HTML 도구가 가장 깔끔하게 인증됩니다.

Q. 내 도구가 너무 사소한 것 같은데 그래도 등록해도 되나요?

네, 등록만 하는 건 부담 없으니 편하게 신청해주세요. DX팀이 보고 "오, 이거 다른 부서도 쓰면 좋겠는데?" 하고 인증 후보로 제안하기도 합니다. 등록 자체가 "이런 노하우가 있다" 알리는 가치가 있어요.

Q. 다른 사람이 만든 도구를 대신 등록해도 되나요?

가능하지만 만든 사람도 알아야 합니다 (담당자 이름이 본인이 아닌 다른 사람으로 등록되기 때문). 이메일이나 카톡 신청 시 "본인 도구가 아니라 [원작자 이름]의 도구입니다"라고 명시해주세요.

Q. 인증받은 후에 다른 매장 매니저들이 어떻게 알게 되나요?

인증 완료 시 ① 도구 허브 카드에 🟢 인증 라벨이 붙고 ② 사내 슬랙·카톡으로 "신규 인증 도구" 안내가 자동 발송됩니다. 매장 매니저들은 도구 허브 URL을 북마크해두고 정기적으로 새 도구를 확인하기도 합니다.

Q. 더 궁금한 점이 있으면?

DX팀 정지훈 (david.jung@ogaren.com) 또는 사내 카톡으로 언제든 물어봐주세요. "이런 도구를 만들고 싶은데 등록 대상이 될까요?" 같은 사전 상담도 환영합니다.
